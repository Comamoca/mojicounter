シンプルな文字数カウンター

<br>

<div align="center">

## 🚀 使い方

- 対象の文字列をページのテキストエリアに貼り付けます。
- テキストエリア下部に文字数などの情報がリアルタイムに表示されます。

## ⬇️  Install

このサイトはPWAに対応しています。
タブをクリックしたりすると「このサイトをインストール」などの文言が表示されるので、それをクリックするとインストールすることができます。

## ⛏️   開発

```sh
bun i
bun run dev
```
